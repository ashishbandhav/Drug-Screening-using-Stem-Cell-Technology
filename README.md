# Drug Screening Using Stem Cell Technology

### üß† Overview
This repository contains the implementation and research findings of **"Drug Screening Using Stem Cell Technology"**, authored by **Bandhakavi Sai Ashish** and collaborators from **Lovely Professional University** and **Guru Nanak Institutions Technical Campus**.

This work integrates **machine learning (ML)** with **stem cell biology** to enhance the precision and efficiency of **drug discovery and toxicity prediction**. By using **hiPSC-derived cardiomyocytes and hepatocytes**, we built a robust drug screening framework that surpasses traditional 2D and animal models in predictive accuracy.

---

### üìä Features
- Utilizes **Support Vector Machines (SVM)**, **Random Forest**, and **CNN** models.
- Performs **toxic vs. non-toxic drug response classification**.
- Analyzes **microscopic cellular images** for phenotype recognition.
- Implements **data preprocessing, feature extraction, and model validation**.
- Achieves over **90% prediction accuracy** on benchmark datasets.

---

### üß© Methodology
1. **hiPSC Differentiation** ‚Üí Generation of cardiomyocytes and hepatocytes.  
2. **Drug Treatment** ‚Üí Exposure of differentiated cells to various compounds.  
3. **Data Acquisition** ‚Üí Gene expression and morphological image collection.  
4. **Preprocessing & Feature Extraction** ‚Üí Normalization and pattern detection.  
5. **Model Training** ‚Üí Using ML algorithms to predict drug toxicity.  
6. **Validation & Result Analysis** ‚Üí Comparing results against known benchmarks.
